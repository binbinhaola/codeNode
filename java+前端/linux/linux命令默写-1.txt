1.cd命令
     cd /命令:切换到根目录
     cd -命令：跳到自己的 home 目录 
     cd ..命令：返回到上一级
2.pwd命令
     pwd命令：显示当前文件夹路径
3.mkdir命令：________
     eg:mkdir -pv /abc/123
     -p  确保目录名称存在，不存在就创建
     -v ________

三、常用命令管理之文件管理-----文件浏览
1.less:分页显示文件内容，操作更详细
     打开xx.log文件 less xxx.log
     -m  显示命令的百分比
     -N 显示每行的行号
     空格键：滚动一行
     b：向上翻一页
     d: 向后翻半页
     u: 向前滚动半页
     v:  使用配置的编辑器编辑文件
     q: 退出less命令

2.tail命令
      功能：用于显示指定文件末尾的内容，不指定文件时，作为输入信息进行处理，常用于查看日志文件。
      -n<行数> 显示文件的尾部 n 行内容
      -f 循环读取
      tail -f ip.log   跟踪文件ip.log的增长情况

四、常用命令之文件管理------文件操作
1.touch:创建文件
2.cp: 复制文件
     cp 文件名 ../ :  复制文件到指定文件夹
     cp 文件名 ../文件名: 复制文件到指定文件夹，并重命名
     cp 文件名 指定目录名: 复制文件到指定文件夹
     cp文件名 新的文件名 复制一个新文件打到当前目录
     cp -r 目录名 指定目录名: 若给出的源文件是一个目录文件，此时将复制该目录下所有的子目录和文件。
3.mv命令
     mv 文件名 指定目录(最好使用绝对路径): 移动文件到指定文件夹
     mv 文件名 指定目录(最好使用绝对路径)/新文件名: 移动文件到指定文件夹，并重命名
     mv 文件名 新文件名 重命名文件
     mv 目录名 指定的目录名 指定目录存在，移动文件夹。目录不存在则重命名文件夹
4.rm命令：删除文件或目录            记住:rm -rf 目录名 (空目录,带子目录的都可以删)
     rm 文件名:   删除文件
     rm -f 文件名: 删除文件不询问
     rm -r  目录名: 删除文件夹
     rm -rf 目录名: 删除文件夹不询问
