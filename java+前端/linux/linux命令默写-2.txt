1.切换目录的命令（3个）
	1):cd .. 返回上一级		
	2):cd /  返回根目录
	3):cd -  返回上一次目录	
----------------------------------------------------
2.列出文件夹下文件列表？（4个）
	1):ls     显示全部文件
	2):ls -l  显示文件显示详细信息,简写(ll)
	3):ls -a  显示全部文件包括隐藏文件
	4):ls -al 显示详细全部文件包括隐藏文件
----------------------------------------------------
3.创建目录和删除目录？（4个）
	1):mkdir 创建单层目录
	2):mkdir -p 创建多层目录
	3):rmdir 删除目录，会有询问
	4):rmdir -rf 递归删除，不会询问
----------------------------------------------------
4.查看文件内容命令？（4个）
	1):cat  查看全部文件内容，一般适合小文件
	2):more 带分页查询，空格的含义：翻页 回车的含义：1行行走
	3):less 带分页查询，且带行号，空格的含义：翻页 回车的含义：1行行走
	4):tail -f 动态加载
	5):head -n 查看文件前n行
----------------------------------------------------
5.创建空文件？（1个）
	1）：touch 文件名
----------------------------------------------------
6.向文件中追加内容？
	1):echo 内容 >> 文件名  例如：echo 'hello'>> aa.txt
	2):cat  文件a >> 文件b  例如：cat aa.txt >> bb.txt
----------------------------------------------------
7.编辑文件，保存文件，退出，强制退出？
	1):vi或者vim 可以编辑文件
	2):wq 退出保存
	3):wq! 强制保存退出
	4):x  保存退出
----------------------------------------------------
8.删除文件？包括常用参数？
	1):rm 文件
	2):参数
		-f：强制删除，忽略不存在的文件，从不给出提示。
		-i：交互模式删除文件，删除文件前给出提示。
		-r：递归的删除目录下面文件以及子目录下文件。
		-R：递归的删除目录下面文件以及子目录下文件。
		-v：显示运行时详细信息
----------------------------------------------------
9.新建用户，设置密码？
	1):useradd 用户名
	2):passwd 用户名
----------------------------------------------------
10.权限
	1):r:读取权限，w:写权限 x:执行权限
	2):权限u=rwx g=rwx o=rwx 分别表示 当前用户，本组用户，其他用户组都有读写执行权限
	3):chmod 777 aa.txt 授予aa.txt文件读写执行的权限
----------------------------------------------------
11.查看进程的命令？
	1):ps -ef|grep -i 名字
	2):ps -aux|grep 名字
----------------------------------------------------
12.杀死进程的命令
	1):kill -9 PID(上面查看进程的命令)

----------------------------------------------------
13.解压命令
	1)：tar -zxvf 压缩包名  -c:指定解压路径
----------------------------------------------------
14.如何查询文件和文字在哪里？
	1):whereis 文件名
	2):find -name 字符名
----------------------------------------------------
15.切换用户
	1):su 普通用户
	2):su root  还要密码
----------------------------------------------------
16.查看磁盘使用情况？查看cpu和内存使用情况？
	1) df
	2) top
----------------------------------------------------
