## 数据类型函数

- `int()`：将一个数值或字符串转换成整数，可以指定进制。括号内输入打印的字符
- `float()`：将一个字符串转换成浮点数。
- `str()`：将指定的对象转换成字符串形式，可以指定编码。
- `chr()`：将整数转换成该编码对应的字符串（一个字符）。
- `ord()`：将字符串（一个字符）转换成对应的编码（整数）。
- `type()`：将一个变量的类型进行返回。



## 打印函数

print函数可以通过%d来直接获取后面变量的值

%.1f 数字可以控制浮点数的小数位数

> %需要通过 `%%`的转义来表示百分号

```python
print('%d + %d = %d' % (a, b, a + b))
print('%d - %d = %d' % (a, b, a - b))
print('%d * %d = %d' % (a, b, a * b))
print('%d / %d = %f' % (a, b, a / b))
print('%d // %d = %d' % (a, b, a // b))
print('%d %% %d = %d' % (a, b, a % b))
print('%d ** %d = %d' % (a, b, a ** b))
```
